# FisharBotX Dashboard

لوحة تحكم تفاعلية لصيانة الأنظمة وتحليل البيانات، مصممة خصيصًا للعمل من الهاتف المحمول.  
تجمع بين أدوات ذكية، تقارير فورية، وتجربة مستخدم سينمائية.

## المميزات
- أدوات صيانة تلقائية تعمل من Termux
- تقارير لحظية بصيغة HTML وواجهة رسومية
- تكامل مع Firebase وGoogle Sheets
- دعم كامل للغة العربية

## المطور
محمد – مهندس أنظمة ومصمم أدوات ذكية من الهاتف المحمول.
